AI IT Helpdesk chatbot that runs locally.

## Setup

1. Create virtual environment and install dependencies:
   `pip install -r requirements.txt`
2. Set API key: copy `.env.example` to `.env` and fill in `GEMINI_API_KEY`.
3. Start backend: `python backend/app.py`
4. Open `frontend/index.html` in browser.
=======
 AI Helpdesk System is an AI-powered platform combining an IT support chatbot and an advanced ChatPro assistant. Users can troubleshoot technical issues, ask general questions, upload files for AI analysis, and manage chats with secure login, conversation history, and search in a clean interface.
