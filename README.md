🤖 ai-helpdesk-system

AI Helpdesk System is an AI-powered platform that combines an IT support chatbot with an advanced AI assistant.  
It helps users troubleshoot technical problems, ask questions, analyze files, and manage conversations with login, chat history, and search — all running locally using Flask and Google Gemini.

 ✨ Features

 🤖 AI IT Helpdesk chatbot for solving technical issues  
 🧠 ChatPro mode for advanced AI conversations  
 🔍 Search past conversations instantly  
 📂 Upload files and get AI analysis  
 🔐 Secure login and signup system  
 🧾 Chat history for each user  
 🌐 Google search powered responses  
 ⚡ Fast local deployment using Flask
